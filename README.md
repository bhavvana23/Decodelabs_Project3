# SQL Ecommerce Sales Analysis

## 📌 Project Overview

This project demonstrates practical SQL data analysis using Microsoft SQL Server. The dataset contains e-commerce sales information, and SQL queries were written to analyze sales performance, product trends, revenue generation, customer purchasing behavior, and referral source performance.

The main objective of this project is to transform raw business data into meaningful insights using SQL.


## 🎯 Objectives

- Analyze total sales and revenue
- Calculate total quantity purchased
- Identify top-performing products
- Find least-selling products
- Analyze referral source performance
- Generate business insights using SQL
- Strengthen SQL querying skills


## 🛠️ Tools & Technologies

- Microsoft SQL Server (SSMS)
- SQL
- Microsoft Excel


## 📂 Dataset Information

The dataset contains e-commerce sales records including:

- Product Name
- Quantity Purchased
- Unit Price
- Total Price
- Referral Source
- Order Details

- MAX()
- Aggregate Functions


## 📈 Business Questions Solved

- What is the total revenue generated?
- What is the total quantity purchased?
- Which products generated the highest revenue?
- Which products had the lowest sales?
- Which referral source generated the highest revenue?
- What is the average quantity purchased for each product?
- Product-wise revenue analysis
- Sales performance analysis
